# MinuteMind 🎤🧠

**TalkBack AI** is an offline-first AI-powered meeting assistant that turns conversations into structured, actionable meeting minutes.

Built with .NET technologies, it captures audio, transcribes speech locally, extracts key insights, and generates clean, exportable summaries — all with a focus on performance and privacy.

---

## 🚀 Features

- 🎤 **Audio Recording & Upload**
  - Record meetings directly in the app
  - Upload existing audio files

- 🧠 **Offline AI Transcription**
  - Speech-to-text powered by on-device Whisper model
  - No internet required for core functionality

- 📝 **Smart Meeting Minutes**
  - Automatically generates:
    - Key discussion points
    - Decisions made
    - Action items

- ✏️ **Editable Output**
  - Review and refine generated minutes before exporting

- 📄 **PDF Export**
  - Download structured, professional meeting summaries

- ⚡ **Offline-First Design**
  - Prioritizes privacy, speed, and reliability

## 🧠 How It Works

1. Record or upload meeting audio  
2. Transcribe speech into text using on-device AI  
3. Process transcript into structured meeting minutes  
4. Edit and refine content  
5. Export as a clean PDF
